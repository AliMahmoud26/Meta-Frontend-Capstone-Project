# Little Lemon Restaurant

This is Little Lemon Restaurant. It was an application released by Meta, I Created it and took it in the frontend capstone project using React Js. I've done this project perfectly and added many other features such as the login page which you can create an account and all your info will saved to the local storage and you can logout whenever you want. Also, on the reservation page, you can customize all of the info you want and it will be saved as well to the local storage.
